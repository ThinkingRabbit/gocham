# 고참 (고마운 참견)
<고참>은 내고민을 남들과 함께 해결할 수 있는 사이트입니다.
내 가벼운 고민들을 올리고, 사람들이 투표할 수 있는 서비스!
유튜브 쇼츠 처럼 가볍게 유저들의 고민을 해결해주는 고마운 참견자가 되보는건 어떨까요?
